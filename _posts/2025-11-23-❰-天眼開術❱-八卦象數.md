**八卦象數（八卦象數療法）** 是一種基於《易經》八卦、中醫五行理論與數字能量對應的調理方法。由中國學者李山玉推廣。

簡單來說，就是**將人體臟腑對應到 1 至 8 的數字，透過「默念」這些數字組合，產生特定的頻率共振，以調節身體氣血。**

以下是整理好的核心對應表、操作規則與範例。

- - -

### 一、 核心對應表 (必須熟記)

這套系統將八卦、數字、五行與人體臟腑對應如下：

|     |       |     |          |         |
| --- | ----- | --- | -------- | ------- |
| 數字  | 八卦    | 五行  | 對應臟腑/部位  | 功能特性    |
| 1   | 乾 (☰) | 金   | 大腸、頭部、骨骼 | 統攝、剛健、寒 |
| 2   | 兌 (☱) | 金   | 肺、口、氣管   | 氣體交換、肅降 |
| 3   | 離 (☲) | 火   | 心、眼、小腸   | 溫暖、明亮、熱 |
| 4   | 震 (☳) | 木   | 肝、足、神經   | 疏泄、爆發力  |
| 5   | 巽 (☴) | 木   | 膽、股、筋    | 風、傳導、優柔 |
| 6   | 坎 (☵) | 水   | 腎、耳、血液   |         |
| 7   | 艮 (☶) | 土   | 胃、手、關節   | 止、凸起、穩固 |
| 8   | 坤 (☷) | 土   | 脾、腹、肌肉   |         |

> **特殊數字「0」**：
>
> * 功能：強化、調和、通經絡。
> * 用法：放在數字前後，用來調整陰陽（奇數為陽、偶數為陰）。

> ### 二、 操作說明
>
> #### 1. 配數邏輯（五行生剋）
>
> 配方通常由一組或多組數字組成，中間用點號 `.` 隔開。
>
> * **相生（補）**：金生水 (1/2 -> 6)，水生木 (6 -> 4/5)，木生火 (4/5 -> 3)，火生土 (3 -> 7/8)，土生金 (7/8 -> 1/2)。
> * **相剋（瀉）**：金剋木，木剋土，土剋水，水剋火，火剋金。
> * **順序**：通常將主要症狀的對應數字放在前面。
>
> #### 2. 默念方式
>
> * **讀法**：一個數字一個數字單獨念，用\*\*普通話（國語）\*\*發音。
>
>   * 例：`650` 讀作「六、五、零」。
>   * 不要讀成「六百五十」。
> * **心態**：放鬆、專注，不要用力思考，類似冥想。
> * **頻率**：默念速度適中，不宜過快或過慢。
> * **時間**：隨時可念，建議每次 15-30 分鐘，睡前或起床時效果較好。
>

> ### 三、 常見配方範例
>
> 以下為針對一般情況的基礎配方（僅供參考，實際狀況因人體質而異）：
>
> #### 1. 失眠、心神不寧
>
> * **配方**：`640.380`
> * **解析**：
>
>   * `640`：6（腎水）生 4（肝木），滋陰養肝。
>   * `380`：3（心火）生 8（脾土），補益心脾。
>   * 整體意圖：心腎相交，安神助眠。
>
> #### 2. 胃痛、消化不良（胃寒型）
>
> * **配方**：`70.80` 或 `380.70`
> * **解析**：
>
>   * `7` 對應胃，`8` 對應脾。
>   * `380`：3（火）生 8（脾土），用火來暖脾胃（去寒）。
>   * 單獨用 `70` 則是直接調理胃氣。
>
> #### 3. 頭痛（高血壓、肝陽上亢型）
>
> * **配方**：`001600` 或 `640`
> * **解析**：
>
>   * `1` 為頭（乾金）。
>   * `6` 為腎水。
>   * `00` (偶數個0) 代表滋陰。
>   * `1600`：金生水，引導頭部的氣血下行歸腎，降低腦壓。
>
> #### 4. 止痛消炎（通用型）
>
> * **配方**：`0001000` 或 `0007000`
> * **解析**：
>
>   * `1` (大腸/乾) 與 `7` (胃/艮) 在象數療法中常用於止痛。
>   * `000` (三個零) 為奇數，偏陽，強化疏通、驅散的效果。
>

> ### 四、 注意事項與禁忌
>
> 1. **好轉反應**：念誦初期若出現打嗝、排氣、發熱、發涼，通常視為氣血運行的正常反應。
> 2. **不適即停**：如果念誦後感到頭暈、噁心、胸悶、心慌，代表該配方**不適合你的體質**（例如火氣大的人誤補了火），請立即停止。
> 3. **非醫療替代**：此方法屬於輔助調理或養生，**急重症、細菌感染、結構性損傷（如骨折）請務必尋求正規醫療**。
